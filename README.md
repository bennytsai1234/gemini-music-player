# Gemini Music Player

這是一個由 Gemini 開發的音樂播放器應用程式。

## 專案結構
此專案分為以下幾個模組：
- **app**: 主應用程式模組。
- **core**: 包含核心功能，如資料儲存、設計系統、解析器、依賴注入和工具函式。
- **data**: 處理資料層邏輯，包括資料庫、資料來源和儲存庫實作。
- **domain**: 定義應用程式的業務邏輯，包括模型、儲存庫介面和用例。
- **player**: 負責音樂播放功能，包括播放器管理、通知和服務。
- **ui**: 包含使用者介面相關的程式碼，如畫面、組件、導航和設定。
